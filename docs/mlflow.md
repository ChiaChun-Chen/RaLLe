# Review the Evaluation Results with MLflow

## 1. Run the MLflow UI

```bash
mlflow ui
```

## 2. Access your localhost

http://localhost:5000

Then you can review the evaluation results.

Example:

e5 (Faiss Flat index, top-5 documents) + Llama2-chat-70B
